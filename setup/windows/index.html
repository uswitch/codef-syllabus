SETUP

Hugo Windows

June 10, 2020

If you’re using a package manager like Chocolatey, it should be present there.

If not, you should be able to follow the official installation directions

What you’ll need to do is

  1. Download a Windows release of Hugo
  2. Make a directory named Hugo in your C: drive
  3. Make a directory named bin in the newly made Hugo diretory
  4. Move the file you downloaded into this bin folder

  1. Unzip the file directly into the bin directory

  1. You should now see 3 files, one of which is hugo.exe

Then you’ll need to make Hugo executable from a terminal, which is something we’ll need later. To do this

  1. Right click your Start button
  2. Click System

  1. Start typing Advanced System Settings into the search bar

  1. Click on the Environment Variables button on the bottom

  1. In the User variables section, find the row that starts with Path.
  2. Double-click on Path

  1. Click the New button
  2. Type in C:\Hugo\bin, press Enter

  1. Click OK at every window to exit
